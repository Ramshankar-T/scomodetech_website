Connecting “Man, Machine and Methods”

A fully configurable middleware IOIT Operating system delivered as a platform, enabling a variety of IOT devices to get connected with standard or customized protocols, codecs and standards with utmost data handling efficiency and accuracy.

SCOMODE is a set of hardware protocols and interfaces which is easily configurable over the cloud to interconnect different type of connected devices. SCOMODE platform is completely modularized covering maximum protocols and interfaces. SCOMODE Enables interconnecting any non standard devices\systems to standard devices and protocols

SCOMODE is a implementation of interconnection of uniquely identifiable objects over the internet with existing infrastructure. The Object can be a Man, Machine or a Method which could be linked to the Internet via standardized or customized protocols and systems. SCOMODE has a integrated Machine learning and Decision making algorithms which could be configured as per the user requirement. It also has a Data analytics like Stochastic , pattern recognition and Big data which can be processed with distributed infrastructure.

Supports multiple CPU platforms like INTEL X86,
X64, ARM 7 onwards, CORTEX, AMD

Can support multiple operating systems like LINUX,
SOLARIS, WINDOWS, WINDOWS CE, UNIX BASED RTOS

Can support various interfaces like UART,
WIFI, BLUETOOTH,
IR, SATA, I2C, SPI, USB, HDMI, AUDIO, FIBER etc.

Supports various standard protocols like
MODBUS, CANBUS, DLMS, RS485, IEC, DNP 3, PLCC etc
with custom protocol wrappers

Supports XML, JSON, CSV, FLAT FILE, MS SQL, MYSQL, MONGO,
POSTRE SQL etc for data handling and logging data
